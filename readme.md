# Projeto Exemplo com Git

Este repositório foi criado para praticar os principais comandos e fluxos de trabalho com Git e GitHub.

## Objetivo

O objetivo deste projeto é aprender e demonstrar como:

- Inicializar um repositório Git
- Criar e acompanhar alterações em ficheiros
- Fazer commits
- Trabalhar com branches
- Enviar alterações para o GitHub
- Atualizar o repositório local com alterações remotas

## Tecnologias utilizadas

- Git
- GitHub
- Markdown
- Linha de comandos / Terminal
